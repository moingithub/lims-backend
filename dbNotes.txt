1- Delete the existing lims database
2- Create new lims database
3- Delete the folder migrations which is under prisma
4. Migrate DB Structure
npx prisma migrate dev --name init
5- Execute script in db from prisma\additional-db-scripts.sql
6. Seed 
npx prisma db seed
or
node prisma/seed.js

sudo pm2 list

sudo pm2 stop 0
rm -rf node_modules package-lock.json
upload changes from fileZilla ( Do not upload node_modules)
npm install
sudo pm2 start 0
